What is this

This repository includes a bunch of useful AUR packages that are not listed in other popular Arch repositories, such as linuxqq-nt-bwrap, wechat-bin, wemeet patches, etc. The repository builder is powered by GitHub Actions. It will automatically pull AUR packages, build them, and release to my GitHub page and Huggingface repository everyday.

Usage

Add the following to your pacman.conf file to enable the OUR repository: 

[our]
Server = https://github.com/Orion-zhen/our/raw/refs/heads/main/$arch
Command line: 
sudo tee -a /etc/pacman.conf << EOF
[our]
Server = https://github.com/Orion-zhen/our/raw/refs/heads/main/$arch
EOF
Then refresh the package database: 
sudo pacman -Syu
